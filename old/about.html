Austin Data Bass Team



Site Description

Our site serves as a database for all of Austin's upcoming concerts. We aim to make finding upcoming concerts a seamless experience by providing all of the venue, artist and event details in one easy to use website. We know how hard it can be to find all the information about one concert in one place. Whether you're an avid concert goer, an artist looking to find a venue or taking your first steps into the world of live music, we have the information you need!

Data & Tools

HTML

Used to create the structure of our webpages.

Bootstrap

Used as a CSS framework to speed up styling.

CSS

Used to style our webpages.

JavaScript

Used to import API data and display it.

Python

Used as our backend language that handles the database.


We used several different RESTful APIs that provided data for our venues, concerts and artist models. Using these disparate data sources allowed us to centralize all the information a concert goer needs to chose the right show! Whether you need Spotify stats on an artist, details about the size and location of a venue or all the details you need to worry about for the show, Austin Data Bass displays it in an easy to read format.
